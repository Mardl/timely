# Usage

As an alias like:
```bash
alias timely="php /<DIR_TO_TIMELY>/timely/bin/timely $*"
```

And use like:
```bash
timely track FOO-127 bar
```

# Commands

## track

### Description

### Syntax

```
timely track <TICKET> <COMMENT>
``

### Options

## show

### Description
Show tracked times

### Syntax

```
timely show [--ft] [<TICKET/PATTERN>]
``

### Options

`--f=<DATE>`

Show from a certain date on


`--t=<DATE>`

Shows a tracked tasks up to a certain date
