```
## show

### Description
Show tracked times

### Syntax

```
timely show [-scf] [<TICKET>]
``

### Options

```
-s
```
Sort by ticket

```
-c
```
List in chronologic order

```
-p
```
Filter by pattern

```
-f
```
Show from a certain date on

```
-t
```
Show up to a certain date
